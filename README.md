# milestone3cmpt-the-coders
Milestone 3 for team the coders

Tools included: 

CSharp: An efficient three-phase code clone detector using modified PDGs 
Tested by: Theo

Srcclone: Detecting code clones via decompositional slicing
Tested by: Theo

GPLAG: detection of software plagiarism by program dependence graph analysis
Tested by: Kayden

Lvmapper: A large-variance clone detector using sequencing alignment approach
Tested by: Trailer swift

Nil: large-scale detection of large-variance clones
Tested by: Quinn

TCore: A novel approach for detecting type-iv clones in test code
Tested by: Leif


We added a CSV and XLSX file for your convenience, they are equivalent feel free to use whichever format is easier
